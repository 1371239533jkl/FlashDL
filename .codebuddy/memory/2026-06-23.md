## mpv 依赖说明

项目使用 `python-mpv`（`import mpv`），依赖系统 `libmpv-2.dll`。卸载 mpv 会报错。可单独保留 dll 文件。

## 四项新功能解锁

### mpv_player.py 新增
- `get_video_tracks()` / `set_video_track()` — 视频轨道切换
- `get_subtitle_tracks()` / `set_subtitle_track()` — 内嵌字幕轨
- `get_chapters()` / `chapter` / `seek_to_chapter()` — 章节导航
- `brightness` / `contrast` / `saturation` / `gamma` (property) + `reset_video_params()` — 画面调整

### player_tab.py 新增
- "视频"按钮：视频轨道菜单 + 亮度/对比度/饱和度/伽马滑块（嵌入式 QSlider in QMenu）
- "章节"按钮：章节列表菜单，显示当前章节 ▶
- "字幕"菜单增强：新增内嵌字幕轨切换
- 控制条新增两个按钮（视频、章节）
